# Moximus Sync RepositoryThis repository stores synchronized Moximus data across devices.## Structure- manifest.json - Repository metadata- devices/ - Device registrations- forges/ - Forge configurations- bindings/ - Device-specific path bindings- threads/ - AI chat threads- workflows/ - Swarm workflows- memories/ - Forge memories## UsageConfigure this repository URL in Moximus Settings > Sync to enable cross-device synchronization.
